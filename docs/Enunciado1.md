# CineVault - Sistema de Gestión de Streaming

La empresa FilmCorp lo ha contratado para desarrollar una aplicación llamada "CineVault" que permitirá a un administrador de contenido gestionar todo lo relacionado con películas y géneros de la plataforma de streaming. El administrador de contenido debe estar pendiente de mantener actualizado el catálogo de películas y la clasificación por géneros. Para iniciar el contrato, la empresa lo cita a una reunión para especificar sus necesidades, como se comenta a continuación.

Al iniciar la aplicación el administrador de contenido verá una pantalla principal, donde podrá ver el logo de FilmCorp, la descripción de la aplicación "CineVault", una opción para agregar película, otra para ir al listado de géneros y el listado de películas disponibles. En el listado de géneros se muestra nombre, descripción, el nùmero de peliculas y año de popularidad y botones para editar o borrar género, siempre y cuando no tenga películas asociadas. En la pantalla de géneros, el administrador de contenido encuentra también una opción para agregar género nuevo y otra para volver a la pantalla principal.

En la opción para agregar película, el administrador de contenido podrá registrar título, director, año de lanzamiento, duración en minutos, sinopsis y seleccionar género de una lista disponible. Desde la pantalla para agregar película el administrador de contenido puede ingresar a una opción para gestionar géneros si necesita agregar uno nuevo.

El listado de películas de la página principal muestra para cada película título, director, año y género, una opción que permite editar toda la información, otra que permite eliminar y una opción ver detalles. La opción de ver detalles permite consultar información completa de la película.

Además, la aplicación debe incluir una funcionalidad de "Reporte de Géneros por Popularidad" que permite al administrador de contenido generar estadísticas de películas por género. En la parte superior de este, deberá mostrar el total de peliculas, el total de categorías y la categoría más popular. El reporte muestra para cada género:  el total de películas y el promedio de duración de las películas. La categoría más popular es aquella con màs películas. En el caso que dos categorias tengan el mimso número de películas la más popular entre ellas es la que tenga el promedio de duración más bajo. El listado debe presentarse ordenado por popularidad.
